# User

## uid

```javascript
user.uid()
```

Returns the uid of the client user as a **String**

## **username**

```javascript
user.username()
```

Returns the username of the client user as a **String**

## discordTag

```javascript
user.discordTag()
```

Returns the discord tag of the client user as a **String**
